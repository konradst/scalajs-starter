# scalajs-starter

## Installation
- install sbt with instructions from https://www.scala-sbt.org/1.x/docs/Setup.html
- ```npm install scalajs-starter```

## Usage
- build Scala to JavaScript
  ```
  npm run build
  ```
- build & execute script
  ```
  npm start
  ```
- test
  ```
  npm test
  ```
- clean build folder
  ```
  npm run clean
  ```

## Features
- Scala.js skeleton (to be used & replaced by CLI)
